# Car Dealer Management System (Introduction to Programming 2)

## 📌 Project Overview
This Python application manages a car dealership, allowing users to add, remove, and track conventional and electric cars.

## 📐 Architecture & Class Hierarchy
- `models/` - Core OOP logic (`Car`, `ElectricCar`, `Dealership`).
- `storage/` - Handles file I/O operations (CSV and TXT).
- `utils/` - Contains validation logic (Regex) and custom decorators.
- `tests/` - Structural unit tests.
